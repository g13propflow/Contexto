# auto-doc

Bitácora de auto-documentación. Por **cada tarea de desarrollo** realizada por Claude
se crea aquí un archivo `.md` con el resumen de lo que se hizo.

## Convención
- Un archivo por tarea, nombrado `YYYY-MM-DD-<slug-de-la-tarea>.md`.
- Basado en [`_PLANTILLA.md`](./_PLANTILLA.md).

## Cada archivo debe incluir
- **Fecha** en que se hizo.
- **Tarea solicitada** en concreto.
- **Rama** en la que se trabajó.
- **Módulo(s) afectado(s)**.
- **Resumen** de lo que se hizo.
- **Decisiones tomadas**.
- **Preguntas y respuestas** (si se le preguntó algo al usuario).
- Si se **tocó trabajo de otros desarrolladores**.
- **Bugs de otros** encontrados y cuáles se resolvieron.
